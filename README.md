# portifolio
Repositório destinado para projeto de portifólio pessoal

Crie um site de portfólio pessoal moderno e responsivo usando HTML, CSS E Bootstrap 5.

Tema visual: "Ministérial"
Paleta de Cores:"Azul e Vermelho"
Estilo do design: "minimalista"

O site deve ter:
-Navbar fixa e responsiva
-Seção Hero com foto, nome e frase de destaque
-Seção "Sobre Mim"
-Seção de projetos com imagens e descrição(2 projetos)
-Seção de experiência ou estudos
-Seção de contatos com fornulário
-Footer moderno com redes sociais

Adicione:
-Animações suaves ao rolar a página
-Efeito hover nos botões e cards
-Ícones usando Bootstrap Icons ou Fount Awesome
-Layout totalmente responsivo para celular e desktop
-Scroll suave entre seções
-Design moderno e profissional

Tecnologias obrigatórias:
-HTML5
-CSS3
-Bootstrap5
-Javascript opcional para interações

Organize os arquivos em:
-index.html
-style.css
-script.js

Use código limpo, organizado e comentado
Crie um visual impressionante e moderno, parecido com sites profissionais de desenvolvedores.
